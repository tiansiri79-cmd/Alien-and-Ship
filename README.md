# Alien-and-Ship

Alien-and-Ship 是基于《Python Crash Course》示例游戏改进而来的版本。本项目使用 Python 与 Pygame 实现，新增了随机外星人、不规则移动、外星人子弹、生命系统、护盾机制等功能，使游戏更加完整与可玩。

---

## 1. 项目简介

本项目是一个 2D 射击游戏，玩家控制飞船躲避并击落外星人。相较原版，本项目加入了以下改进：

- 外星人会从随机位置出现并进行不规则移动  
- 外星人从游戏开始就会发射子弹  
- 独立护盾系统，可激活保护圈抵消一次伤害  
- 生命值系统与自动保存的最高分  
- 自由方向移动（WASD/方向键）  
- 更完善的界面显示（分数、生命、护盾状态）

---

## 2. 游戏特色

### 随机外星人设计
- 外星人从屏幕上方的随机位置生成  
- 拥有随机的水平与垂直漂移行为  
- 小概率改变移动方向  
- 外星人会随机发射子弹  

### 生命系统
- 初始生命为 3  
- 只有在碰到外星人或被外星人子弹击中时才会失去生命  
- 外星人到达底部不会扣生命（仅被移除）  
- 生命为 0 时展示 “Game Over”  

### 护盾系统（独立模块 `shield.py`）
- 按下 T 键启动护盾  
- 护盾可持续 15 秒  
- 护盾能够抵消一次外星人或子弹伤害  
- 护盾被触发或时间到会消失  
- 每 30 秒自动恢复 1 次护盾次数，最多可储存 2 次  
- 激活时会在飞船外绘制蓝色保护圆  

### 分数系统
- 击落外星人获得积分  
- 最高分保存至 `high_score.json`，下次启动自动读取
